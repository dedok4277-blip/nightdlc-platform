Расположение: Левый нижний угол ClickGUI
Размер: 160x40 пикселей
Фон: Темно-серый (35, 35, 40) с округлением 5px

┌────────────────────────────────────────┐
│  ┌────────┐                            │
│  │        │  Username                  │
│  │ Avatar │  (обрезается если длинный) │
│  │ 32x32  │                            │
│  └────────┘                            │
└────────────────────────────────────────┘

Элементы:
1. Аватарка (32x32px, округленная 4px)
   - Если загружена: отображается с сайта
   - Если нет: серый фон с иконкой "U"

2. Никнейм (справа от аватарки)
   - Шрифт: SFP_MEDIUM, размер 8
   - Цвет: светло-серый (200, 200, 200)
   - Автоматически обрезается с "..." если длинный

╔════════════════════════════════════════════════════════════════╗
║                      Поток данных                              ║
╚════════════════════════════════════════════════════════════════╝

1. Запуск клиента
   └─> NightDLC.java инициализация
       └─> WebsiteAPI.loadUserData(uid)
           └─> HTTP GET https://nightdlc.site/api/user/{uid}
               └─> Получение JSON: {username, uid, isAdmin, avatarUrl}
                   └─> Загрузка аватарки (если есть)
                       └─> Конвертация в Minecraft текстуру

2. Открытие ClickGUI
   └─> ClickGuiScreen.render()
       └─> Panel.render()
           └─> Panel.renderUserProfile()
               └─> Отрисовка аватарки + никнейма

╔════════════════════════════════════════════════════════════════╗
║                    Технические детали                          ║
╚════════════════════════════════════════════════════════════════╝

Классы:
- Panel.java: метод renderUserProfile()
- WebsiteAPI.java: загрузка данных
- ClickGuiScreen.java: основной экран GUI

Шейдеры:
- RenderUtil.Texture.customRound() для округленной аватарки
- ShaderType.TEXTURE для текстурного рендеринга

API:
- Эндпоинт: GET /api/user/:uid
- Формат: JSON
- Авторизация: не требуется (публичный)

База данных:
- Таблица: users
- Поля: uid, username, avatar_url, is_admin
